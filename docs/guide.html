Why two-stage filtering?

The free triage pass rejects obviously bad matches before they reach the Claude API — so you only pay to score jobs that have a real chance of matching. A typical run scans 2,000+ jobs but only sends a few hundred to Claude.

Claude Scoring

Every job gets a semantic fit score from 0–100 with detailed reasoning.

What Claude evaluates

  • Role alignment with your target titles
  • Skills overlap with your competencies
  • Seniority match (target / stretch / below)
  • Salary signal vs. your floor
  • Innovation signal (high / medium / low)
  • Location & remote compatibility
  • Network connections at the company

What you get back

  • fit_score — 0 to 100
  • reasoning — 2-3 sentence explanation
  • key_overlaps — why you're a fit
  • key_gaps — what to address
  • salary_signal — above/below your floor
  • seniority_match — target/stretch/below
  • network_connections — who you know there
Network Boost

When you have LinkedIn connections at a company, Claude sees them before scoring. Borderline jobs (score 55–70) get a 3–5 point boost because a warm intro changes the calculus.

Getting Set Up

Six things to configure. Claude Code can walk you through all of them.

1

Clone & install

git clone https://github.com/hasmatt1066/jobsearchtool.git cd jobsearchtool python -m venv .venv && source .venv/bin/activate pip install -e .
2

Add your API keys

Copy the example and paste in your keys.

cp .env.example .env # Then edit .env with your keys: ANTHROPIC_API_KEY=sk-ant-... # console.anthropic.com SERPER_API_KEY=abc123... # serper.dev (free tier)
3

Create your profile

This is the most important file — it defines who you are. Claude scores every job against this.

cp config/profile_index.example.json config/profile_index.json # Edit with your name, target titles, skills, experience, etc.

Include: name, location, salary floor, target titles, core competencies, tools, education, experience highlights, and industry interests.

4

Set a reference job

Find a real job posting that represents your dream role. Paste the full description. Claude uses this as a calibration anchor.

cp config/verisk_reference.json.example config/verisk_reference.json # Paste your gold-standard job posting inside
5

Add LinkedIn connections

Export from LinkedIn: Settings → Data Privacy → Get a copy of your data → Connections. Place the CSV at:

data/connections.csv

This enables the network matching engine — you'll see when you have connections at companies with open roles.

6

Tune your settings

Edit config/settings.json to match your search:

"location": "Denver, CO", # Your city "is_remote": true, # Remote preference "score_threshold": 60, # Min score for reports (0-100) "hours_old": 72, # How recent postings must be "max_jobs_per_run": 500 # Cap on Claude API calls

Data You Provide

Three personal files that stay local and never get pushed to GitHub.

👤

profile_index.json

Your professional identity: name, salary floor, target titles, skills, experience highlights, education. Claude compares every job against this.

⭐

reference job

A real job posting that represents your ideal role. Full description, key themes, salary range. Claude uses this to calibrate what "perfect" looks like for you.

🤝

connections.csv

Your LinkedIn connections export. The tool matches company names to flag when you know someone at a company with an open role.

Privacy note

All three files are in .gitignore. Your profile, connections, search results, and API keys never leave your machine unless you explicitly share them.

CLI Commands

Five commands. The main one is jobsearch run.

CommandWhat it does
jobsearch run Full pipeline — discover, triage, score, report. This is the daily driver.
jobsearch search "query" Ad-hoc single search. Use when you want to explore a specific niche.
jobsearch status Dashboard view — jobs seen, top queries, network stats.
jobsearch add-company "Name" --ats greenhouse --token slug Add a company to ATS monitoring so its career page gets polled.
jobsearch expand-ats /path/to/LinkedInExport Auto-detect ATS platforms from your LinkedIn data and add them.

What to Do with Outputs

The report is the starting point, not the finish line.

🎯

Prioritize by score

90+ scores are rare and worth immediate attention. 75-89 are strong. Below 60 is filtered out automatically.

🤝

Activate your network

When a report shows connections at a company, reach out for a warm intro before applying cold. This is the highest-leverage action.

📝

Use the reasoning

Claude's "key overlaps" and "gaps" are a cheat sheet for tailoring your cover letter and interview prep for each role.

💰

Check salary signals

The salary_signal field tells you if a role is likely above or below your floor — even when the posting doesn't list compensation.

🔄

Iterate your profile

If great roles score low, your profile might need tuning. Ask Claude Code to update your target titles, competencies, or experience highlights.

📈

Track query performance

Run jobsearch status to see which search queries find the best jobs. Double down on what works, prune what doesn't.
